Train Model¶

This runs Step 1 of The Cannon: uses the training set to solve for the best-fit model.

TheCannon.train_model.do_one_regression(lams, fluxes, ivars, lvec)[source]¶

Optimizes to find the scatter associated with the best-fit model.

This scatter is the deviation between the observed spectrum and the model. It is wavelength-independent, so we perform this at a single wavelength.
